Phabricator

Phabricator is a suite of open source tools for peer code review, task management, and project communication.

We have our very own instance running at 10.4.3.65

You'll need to be inside the campus network to access it. If you're not on campus, use the VPN.

Getting started

Get an account on github

If you don't have one already, get an account on GitHub at
github.com/join

Get an account on Phabricator

Navigate to 10.4.3.65 and register using your IIIT LDAP email address (eg. "someone@students.iiit.ac.in") and password.

Pick a nice username! LDAP usernames are generally boring.

You'll have to wait for approval by an administrator once you register. Sorry about that.

Look for Good-First-Tasks

We have marked some tasks on phab as "Good first tasks".

You can find them by searching in Maniphest.

Claim a task

Once you find a task you like (and someone else hasn't claimed it), claim it!
